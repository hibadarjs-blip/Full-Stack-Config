# Full Stack Conf

A website for Full Stack Conf, a one-day JavaScript conference.

## Overview

Full Stack Conf is an informational website built with HTML, CSS, and Bootstrap. The site welcomes visitors and invites them to sign up for more information about the upcoming JavaScript conference.

## Features

- **Responsive Design**: Built with Bootstrap for mobile-friendly experience
- **Email Sign-up**: Visitors can subscribe to receive conference updates
- **Multi-page Layout**: Includes main landing page and secondary pages
